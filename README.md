# Image Processing Practice
(Practical Implementation of Image Processing.)
This repository contains practice exercises for **Image Processing** using Python and popular libraries like **NumPy**, **Matplotlib**, and **scikit-image**. The goal is to explore fundamental image processing techniques and understand how to manipulate images programmatically.

---

## Notebooks

### 1. [Practical_1A.ipynb](https://github.com/AnjaliYadav-04/Image_Processing_Practice/blob/main/Practical_1A.ipynb)
- Introduction to image reading and visualization.
- Performing basic operations on images like grayscale conversion, pixel manipulation, and visualization.
- Using `skimage` and `matplotlib` for image handling.

### 2. [Practical_1B.ipynb](https://github.com/AnjaliYadav-04/Image_Processing_Practice/blob/main/Practical_1B.ipynb)
- Advanced image processing techniques.
- Color space conversions (e.g., RGB to HSV).
- Applying filters and transformations.
- Understanding image histograms and adjustments.

## How to Run

1. Clone the repository:

git clone https://github.com/AnjaliYadav-04/Image_Processing_Practice.git
cd Image_Processing_Practice
Install required libraries:

pip install numpy matplotlib scikit-image
Open Jupyter Notebook:

jupyter notebook
Run Practical_1A.ipynb and Practical_1B.ipynb to explore image processing exercises.
